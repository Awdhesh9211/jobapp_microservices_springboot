STEP 1:  build jar without Test
      ./mvnw clean package -DskipTests

STEP 2:  Test it Working
      java -jar target/companyms-0.0.1-SNAPSHOT.jar

STEP 3:  Create a DockerFile
      '''
      # ---- Base image (lightweight + stable)
      FROM eclipse-temurin:17-jdk-jammy

      # ---- App directory
      WORKDIR /app

      # ---- Copy jar (make sure jar already built)
      COPY target/*.jar app.jar

      # ---- Expose application port
      EXPOSE 8081

      # ---- Run app
      ENTRYPOINT ["java","-jar","/app/app.jar"]
      '''

STEP 4: Build Image
   PS D:\SPRING_BOOT_PROJECTS\microservices\companyms> docker build -t awdhesh9211/companyms:1.0 .
